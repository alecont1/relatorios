Modo Rapido
Checklist ja vem pre-preenchido com OK em todos os itens. Voce so precisa adicionar as fotos e ajustar o que nao estiver conforme.
Todos os itens marcados como OK
So adicionar fotos obrigatorias
Alterar itens nao-conformes se necessario
~5 minutos para completar
Modo Completo
Preencha cada item do checklist manualmente, um por um. Ideal para vistorias detalhadas ou quando ha muitos itens nao-conformes.
Controle total sobre cada item
Adicionar comentarios detalhados
Fotos por item especifico
~15-20 minutos para completar
